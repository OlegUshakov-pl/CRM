Задание: переработать интерфейс чата в Django CRM.

1. Убрать AI Assistant из файла templates/base.html

2. В файле sidebar.html (или где расположен левый сайдбар) добавить новую кнопку в самом низу:
   - Иконка sparkles цвет text-indigo-500
   - Кнопка должна открывать модальное окно чата.

3. Создать отдельный шаблон chat_modal.html, содержащий:
   - модальное окно поверх всех элементов (z-index высокий)
   - возможность перемещать окно мышью (drag)
   - возможность изменять размер окна (resize: both)

4. Добавить JS‑функционал:
   - drag & drop для модального окна
   - resize через CSS + JS
   - кнопка закрытия (X)
   - кнопка сворачивания (—)

5. Подключить модальное окно в base.html через include:
   {% include "chat_modal.html" %}

6. Убедиться, что чат работает на всех страницах, но не занимает место в Dashboard.

7. Стиль модального окна:
   - bottom: 20px
   - right: 20px
   - min-width: 350px
   - min-height: 400px
   - background: по теме сайта
   - border-radius: 0px
   - box-shadow: 0 4px 20px rgba(0,0,0,0.2)
   - overflow: hidden

8. Ничего не менять в логике чата, только UI.
